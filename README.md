# 📰 Fake News Detector
A Machine Learning web application built using Streamlit that predicts whether a news article is **Fake** or **Real** using Natural Language Processing techniques.

---

## 🚀 Features

- Detects whether entered news text is fake or real  
- Uses TF-IDF Vectorization for text preprocessing  
- Logistic Regression model for classification  
- Interactive Streamlit web interface  
- Real-time prediction on user input  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- Scikit-learn  

---

## 🤖 Machine Learning Concepts Used

- Natural Language Processing (NLP)  
- Text Vectorization (TF-IDF)  
- Binary Classification  
- Logistic Regression

## ▶️ How to Run Locally

1. Clone the repository  

```bash
git clone https://github.com/your-username/fake-news-detector.git
````

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run Streamlit app

```bash
streamlit run app.py
```
